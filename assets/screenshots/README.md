# Screenshot Guide

Use these filenames in the project root README:
- `assets/screenshots/gui-pack.png`
- `assets/screenshots/gui-extract.png`
- `assets/screenshots/gui-list.png`
- `assets/screenshots/gui-verify.png`

Suggested capture tips:
- Use release build (`cargo run --release --features gui --bin noer22_gui`).
- Capture at ~1366x768 or higher.
- Hide sensitive paths/passwords before publishing.
